# Output Documents Catalog

This directory contains final reports, deliverables, and workshop materials.

## Directory Structure

```
output-documents/
├── data_atlas_2025.pdf         # Final data atlas (main deliverable)
├── workshop_reports/           # Workshop materials and summaries
│   ├── 2025_02_01_workshop/
│   └── maps/
└── technical_reports/          # Technical documentation and methods
```

## Documents

### Data Atlas
- **File**: `data_atlas_2025.pdf`
- **Description**: Comprehensive atlas of biodiversity data and conservation priorities for Bowen Island
- **Contents**:
  - Executive summary
  - Species distributions (193 species)
  - Habitat maps (~50 layers)
  - Conservation priority areas
  - Threat assessments
  - Protected area gaps
  - Recommendations for action
- **Pages**: ~100-150 pages
- **Generated by**: `analyses/08_data_atlas.qmd` → Quarto PDF output
- **Status**: Updated 2025-05-07

### Workshop Reports

#### February 1, 2025 Workshop
- **Folder**: `workshop_reports/2025_02_01_workshop/`
- **Contents**:
  - Workshop summary PDF
  - Presentation slides
  - Maps used during workshop
  - Participant feedback
  - Initial Zonation results
- **Audience**: Caring4Nature Group, community stakeholders

#### Workshop Maps
- **Folder**: `workshop_reports/maps/`
- **Contents**: Print-ready maps for community presentations
- **Format**: Large-format PDF (11x17", 24x36")
- **Types**:
  - Conservation priority maps
  - Species richness overlays
  - Threat assessments
  - Gap analysis

### Technical Reports

#### Methods Documentation
- **File**: `technical_reports/methods_report.pdf`
- **Description**: Detailed methods for SDM processing, Zonation analysis, data integration
- **Audience**: Researchers, consultants, technical reviewers

#### Data Quality Report
- **File**: `technical_reports/data_quality_report.pdf`

  - Recommendations for final analysis

## Document Generation

### Rendering Documents

```r
# Render data atlas
quarto::quarto_render("analyses/08_data_atlas.qmd")

# Render specific workshop materials
quarto::quarto_render("analyses/archive/feb1_maps.Rmd")
```

### PDF Output Settings
- **DPI**: 300 for figures
- **Page Size**: Letter (8.5" x 11") or custom
- **Margins**: 1" all sides
- **Font**: Arial or Helvetica (accessibility)
- **Color**: Full color for screen, grayscale option for print

## Version Control

Documents are versioned by date in filename:
- `data_atlas_2025.pdf` (final version for 2025)
- `data_atlas_20250507.pdf` (version from May 7, 2025)
- Keep previous versions for reference

## Updates

To update documents:
1. Edit source Quarto files in `analyses/`
2. Update figures in `output-figures/`
3. Re-render with `quarto render`
4. Copy output PDF to this directory with versioned filename
