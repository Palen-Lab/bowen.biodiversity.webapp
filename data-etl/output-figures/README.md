# Output Figures Catalog

This directory contains all generated maps and visualizations from the biodiversity analysis.

## Directory Structure

```
output-figures/
├── species/            # Species distribution maps
├── habitats/           # Habitat extent and richness maps
├── conservation/       # Conservation priority maps
├── threats/            # Threat assessment maps
├── protected_areas/    # Protected area analysis maps
└── atlas/              # Data atlas figures (final publication quality)
```

## Figure Organization

### Species Maps (`species/`)
- Individual species distribution maps
- Species richness by taxon (mammals, birds, reptiles, amphibians)
- Red-listed and Blue-listed species maps
- Species hotspot analysis
- **Format**: PNG, PDF (high-resolution)
- **Generated by**: `analyses/02_input_species.qmd`

### Habitat Maps (`habitats/`)
- TEM ecosystem type distributions
- Forest age class extent
- Wetland types and locations
- Stream and riparian buffers
- Coastal and intertidal habitats
- Habitat richness composites
- **Format**: PNG, PDF
- **Generated by**: `analyses/03_input_habitats.qmd`

### Conservation Priority Maps (`conservation/`)
- Zonation rankmap (continuous 0-1 priority)
- Top 10%, 20%, 30% priority areas
- Priority areas overlaid with:
  - Protected areas
  - Private land
  - Human disturbance
- Gap analysis maps
- **Format**: PNG, PDF
- **Generated by**: `analyses/04_conservation_values.qmd`

### Threat Maps (`threats/`)
- Development pressure overlays
- Wildfire risk (WUI) zones
- Human disturbance gradient
- Climate change vulnerability
- Threat × Conservation priority intersections
- **Format**: PNG, PDF
- **Generated by**: `analyses/05_threats_development.qmd`, `analyses/06_threats_fire.qmd`

### Protected Areas (`protected_areas/`)
- Current protected area extent
- Protected area types (parks, ALR, covenants, etc.)
- Gap analysis (priority areas not protected)
- Potential new protected areas on public land
- **Format**: PNG, PDF
- **Generated by**: `analyses/07_protected_areas.qmd`

### Data Atlas (`atlas/`)
- Final publication-quality figures for data atlas
- All maps with consistent styling
- Multi-panel figures
- Summary infographics
- **Format**: High-resolution PNG, PDF, EPS
- **Generated by**: `analyses/08_data_atlas.qmd`

## File Naming Conventions

```
{category}_{description}_{date}.{ext}

Examples:
- species_ursus_americanus_distribution_20250121.png
- habitat_TEM_richness_20250121.pdf
- conservation_top30pct_20250121.png
- threat_development_overlay_20250121.pdf
```

## Map Specifications

### Basemap
- Bowen Island boundary (bold outline)
- Ocean/water (light blue)
- Road network (gray)
- North arrow and scale bar
- Coordinate grid (optional)

### Color Palettes
- **Conservation Priority**: Yellow → Green gradient (low → high)
- **Species Richness**: Viridis or Plasma
- **Threats**: Red gradient (low → high risk)
- **Habitats**: Categorical (colorblind-friendly)

### Resolution
- **Web**: 300 DPI PNG
- **Print**: 600 DPI PDF/EPS
- **Screen**: 150 DPI PNG

## R Code for Generating Maps

All figures use the custom plotting functions from `R/plots.R`:

```r
library(bowen.biodiversity.webapp)  # Load plotting functions
library(terra)

# Example: Conservation priority map
priorities <- rast("output-data/zonation/rankmap.tif")
bowen_map(
  raster = priorities,
  title = "Conservation Priority",
  legend_label = "Priority (0-1)",
  palette = "YlGn",
  output = "output-figures/conservation/priority_map.png"
)
```

See `R/plots.R` for full documentation of plotting functions.

## Figure Updates

To regenerate figures:
1. Ensure output data is current (`output-data/`)
2. Render relevant Quarto document in `analyses/`
3. Figures automatically saved to this directory
