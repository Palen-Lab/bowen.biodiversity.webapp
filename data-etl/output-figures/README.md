# Output Figures

Generated maps and visualizations. Most files are not committed to git due to size — they are also synced to Google Drive via `ggsave_drive()`.

## Directory Structure

```
output-figures/
└── data-atlas/
    ├── annotated/      # Maps with north arrow, scale bar, and title
    └── unannotated/    # Base maps without annotation (for layout use)
```

## Regenerating Figures

All map targets are defined in `_targets.R` and generated by functions in `R/pipeline-create-plots.R`.

```r
targets::tar_make()  # Regenerates only outdated figures
```

To force a specific figure to regenerate:
```r
targets::tar_invalidate(fire_index_plot_annotated)
targets::tar_make()
```

## Map Style

All maps use a shared template (`template_plot()` in `R/utils-plots.R`) with:
- Maptiler "Backdrop" basemap (fetched once as the `basemap` target)
- Ocean mask clipped to Bowen Island shoreline
- Roads and trails overlay
- Consistent legend position (top-left)
